# Project Title
Amir Qureshi Level Design Portfolio

## Project Description
I made a portfolio to show off my level design projects that I've previously done and to put any more that I may do in the future. I also wanted to include a short bio and a contact link.

## Intended User or Player
This interface is designed for employers who are looking through applicants for game design roles.

## User / Player Goal
They should be able to easily go through the page and find information about my projects, and some short info on me.

## Inspiration Interfaces
1. Nathalie Jankie level design portfolio
The main thing I took from this portfolio was the idea for having an image in the header of the page. I think it creates good visual interest right when you load up the page.

2. David shaver level design portfolio
The main inspiration I took from this page was the rectangular images of projects. I really liked how that looked layout wise and implemented something similar.

## Design Choices
- Layout: I made sure that the layout of the page was simple and went straight down, making it easy to follow. 
- Color: I tried choosing images from my projects that had striking blues and green tones to have a coherent color scheme across the page.
- Typography: I used the same font for most of the information sections, but used a different font for the navbar at the top to differentiate them.
- Visual hierarchy: The visual hierarchy is shown with shadow lines that come from the different boxes created with CSS.
- Feedback states: The boxes used to create the site will shrink as the page moves. This helps to make the page readable on all sorts of devices with different screen sizes.
- Responsive design: When you hover over the images, they will turn a lot more transparent to indicate that you can click them. The navigation bar will also turn red when you hover over them.
